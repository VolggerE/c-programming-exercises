# C Programming Exercises

This repository contains small C programming exercises completed as part of my learning process in programming and computer science fundamentals.

The focus is on basic concepts such as recursion, arrays, strings, loops, and simple algorithmic problem solving.

## Contents

- recursion_exercises.c → recursive functions (sum, factorial, sequence, digit sum)
- count_letters.c → counts alphabetic characters in a string
- monotonic.c → checks whether an integer array is monotonic (increasing or decreasing)

## Purpose

These exercises were created to strengthen my understanding of C programming fundamentals, problem solving, and algorithmic thinking.
